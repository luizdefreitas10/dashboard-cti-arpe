# Soluções Digitais — imagens e planilha

## Imagens dos cards

Coloque os arquivos (PNG, JPEG, WebP) nesta pasta. Na planilha **SOLUCOES-DIGITAIS-CTI.xlsx**, coluna **IMAGEM**, use **apenas o nome do arquivo** (ex.: `dashboard-cti.png`), igual ao arquivo aqui.

O site serve em `/solucoes-digitais/nome-do-arquivo.ext`.

Também é possível usar **URL completa** `https://...` na coluna IMAGEM.

## Planilha

O arquivo de carga pode ficar aqui para referência (`SOLUCOES-DIGITAIS-CTI.xlsx`) ou em outro lugar; o upload é feito pela página **Importar Dados** no app.

Ver especificação: `docs/solucoes-digitais-prompt-implementacao.md`.
