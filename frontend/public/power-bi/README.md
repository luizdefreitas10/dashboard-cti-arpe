# Miniaturas dos dashboards Power BI

Coloque aqui as imagens (prints) dos relatórios para exibir nos cards da tela **Power BI**.

## Como vincular na planilha

Na planilha `Links Dashboards.xlsx`, use a coluna opcional **IMAGEM** com **apenas o nome do arquivo**, por exemplo:

| NOME | LINK | ... | IMAGEM |
|------|------|-----|--------|
| Dashboard Normatização | https://... | ... | `normatizacao.png` |

O front buscará o arquivo em `/power-bi/normatizacao.png` (esta pasta).

Também é possível usar uma **URL completa** na coluna IMAGEM (CDN, SharePoint público, etc.).

## Formatos sugeridos

WebP ou PNG, proporção larga (ex.: 16:9), para caber bem no card.
